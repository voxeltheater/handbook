# ETC ColorSource PAR

- Wash Light
- RGBL
- Circle and Oval lenses available
- 5-Pin DMX In/Out
- PowerCON In/Out

### DMX Personality
When using QLab as the lighting console, we recommend operating the ColorSource PARs in 6-channel Direct Mode.

| Address | Function  |
| ------- | --------- |
| 1       | Intensity |
| 2       | Red       |
| 3       | Green     |
| 4       | Blue      |
| 5       | Lime      |
| 6       | Strobe    |
